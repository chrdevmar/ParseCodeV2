My Computer Support Check In App
My Computer Support is a computer repair business. The MYCS Check in app replaced paper job forms and made it easy for staff to track, update and contribute to current repairs. This web/mobile application was built using AngularJS and Ionic. All data relating to the clients and the jobs was stored in a firebase database which was authenticated with a security token within the application.
The application is able to process and display lots of data stored in the database, this data is editable through the app with the use of Angular. The Ionic framework allowed me to make the app responsive and highly interactive. The app is usable on android, iOS and desktop devices.
This was my first real experience building a data-driven mobile application. I was thrown in the deep end of the Angular pool and was able to swim thanks to my solid understanding of general programming concepts.
